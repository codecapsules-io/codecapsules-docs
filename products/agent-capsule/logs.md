# Logs

Use Capsule logs to monitor activity and debug issues.

## View Capsule Logs

View a Capsule's system logs in the **Logs** tab.

## View Access Logs

For Capsules that serve web traffic, open the **Access Logs** tab to view HTTP request logs.
